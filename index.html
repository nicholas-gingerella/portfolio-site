My Mission

To reduce the clutter and complexity behind big systems, I focus on creating logical abstractions built on a foundation of clean, readable code for us mere mortals to understand. Using clean code and solid engineering processes, my goal is to bring these benefits to the projects I work on:

Maintainability

Maintainable software reduces time spent working through needlessly complex logic.

Adaptibility

Clean code creates software that is faster to change in order to address the shifting needs of the customer

Reliability

Well Maintained software is easier to test, easier to fix, and harder to break.
